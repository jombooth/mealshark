# Mealshark Privacy Policy

Effective date: May 15, 2026

Mealshark is a Chrome extension that helps MealPal users compare available
meals by discount, credit value, newly added meals, cost in dollars, and
map location directly on MealPal lunch and dinner pages.

Mealshark is an independent extension and is not affiliated with, endorsed by,
or sponsored by MealPal.

## Data Mealshark Processes

Mealshark runs only on `https://secure.mealpal.com/*`. On supported MealPal
lunch and dinner pages, it processes MealPal page content and MealPal API
responses in your browser to provide its comparison features.

This may include:

- meal names, descriptions, images, restaurants, addresses, and map coordinates
- meal credit prices, retail prices, discounts, and newly added meal flags
- MealPal plan pricing fields used to compute the cash value of credits
- the current MealPal page URL and page title
- Mealshark UI preferences, such as sidebar state, result count, filters, and sort mode

## Local Processing

Mealshark processes MealPal data locally in your browser. Mealshark does not
send MealPal menu data, account data, pricing data, browsing activity, or
extension settings to a Mealshark server.

Mealshark does not use analytics, advertising SDKs, tracking pixels, or
third-party telemetry.

## Local Storage

Mealshark uses Chrome extension storage to remember local preferences and the
last MealPal page detected by the extension. Stored settings may include:

- whether the sidebar is open or closed
- the number of results to display
- whether the newly added meal filter is enabled
- the selected sort mode
- the last MealPal page URL, title, and capture time

Mealshark does not store the full MealPal menu response in Chrome storage.
Captured meal data is used in memory while the MealPal page is open.

You can remove locally stored Mealshark data by removing the extension or by
clearing extension site data through Chrome.

## Data Sharing

Mealshark does not sell, rent, or share user data.

Mealshark does not transfer user data to third parties.

## Permissions

Mealshark requests these Chrome extension permissions:

- `storage`: saves local extension preferences in your browser.
- `https://secure.mealpal.com/*`: lets Mealshark run on MealPal pages so it can
  read MealPal menu information, add comparison UI, annotate meal tiles, and
  interact with the MealPal map.

## Contact

For privacy questions, use the support URL listed on Mealshark's Chrome Web
Store listing or open an issue in the public Mealshark GitHub repository.
